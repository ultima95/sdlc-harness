# Backlog — deferred work

Durable, git-versioned list of work intentionally **not** built when it wasn't needed
(YAGNI). Spec & Plan appends here instead of implementing speculative scope; Intake reads
this when starting a new task so deferred ideas resurface as candidate work.

This file is **never** regenerated by `/sdlc memory-refresh` or Phase 0 — it is the one place
deferred ideas survive after a task reaches `done`.

## Format

One line per item:

```
- [ ] YYYY-MM-DD · from <branch-or-task> — <what> (why deferred)
```

Check an item off (`- [x]`) or delete it when it's picked up or dropped.
